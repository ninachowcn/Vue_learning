绑定 HTML Class -- 组件

当在一个自定义组件上使用 class 属性时，这些类将被添加到该组件的根元素上面。这个元素上已经存在的类不会被覆盖。

my-component class="active" 字体颜色红色

template:'< p class="error">p标签中添加背景颜色蓝色< /p>'

两个class不会相互覆盖


带绑定class的用法

my-component2 v-bind:class="{active: isActive}", data中 isActive : true

template:'< p class="error">p标签中添加背景颜色蓝色22222< /p>'
