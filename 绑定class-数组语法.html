绑定 HTML Class -- 数组

1, 普通数组绑定

基本语法：v-bind:class="[ activeClass, errorClass]"，参数为键值对名称

data : {activeClass : 'active',errorClass : 'error'}，键值为class名字

div 被渲染为 < div class="active, error" >

字体颜色红色生效，背景蓝色生效

2，在数组中添加表达式

v-bind:class="[ isActive ? activeClass:'', errorClass]"

isActive=false时，字体颜色红色失效，否则生效，背景蓝色始终生效

3，在数组语法中用对象语法

v-bind:class="[{ active:isActive }, bgcClass]

字体颜色红色生效，背景蓝色始终生效
